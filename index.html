Learn Programming

JavaScript

JavaScript adalah salah satu bahasa pemrograman tingkat tinggi yang bersifat client-side. JavaScript merupakan salah satu teknologi inti dalam pengembangan sebuah web selain HTML dan CSS. JavaScript biasanya digunakan dalam membuat halaman web yang interaktif dan merupakan bagian penting dalam fungsionalitas sebuah web application.

Javascript sangat populer di internet dan dapat berjalan di sebagian besar web browser populer seperti Google Chrome dan Mozilla Firefox. Dalam kurun waktu 20 tahun terakhir JavaScript menjadi salah satu bahasa pemrograman yang paling banyak digunakan di seluruh dunia.


Sejarah

JavaScript pertama kali dikembangkan oleh Brendan Eich dari Netscape yang awalnya diberi nama LiveScript pada September 1995. Pada tahun 1996, JavaScript secara resmi disebut sebagai ECMAScript. JavaScript terus dikembangkan oleh komunitas developer dari European Computer Manufacturers Association (ECMA). Pada tahun 1998, ECMAScript 2 resmi diluncurkan dan ECMAScript 3 mulai diperkenalkan pada tahun 1999.

Sampai sekarang Javascript terus dikembangkan hingga bisa digunakan di berbagai perangkat seperti mobile device. Telah tercatat di tahun 2016 bahwa 92% website di seluruh dunia menggunakan JavaScript.

Python

Python adalah sebuah bahasa pemrograman yang dapat melakukan eksekusi sejumlah instruksi multiguna secara langsung (interpretatif) dengan metode orientasi objek. Filosofi perancangan yang berfokus pada tingkat keterbacaan kode menjadikan Python dapat digunakan untuk berbagai keperluan dalam pengembangan aplikasi perangkat lunak dan mampu berjalan di berbagai sistem operasi yang ada saat ini.

Python mampu menjadikan program dengan skala sangat rumit menjadi mudah. Karena struktur dan syntax nya yang rapi dan mudah dipahami menjadikan Python relatif lebih mudah untuk dipelajari serta memiliki dukungan komunitas yang besar.


Sejarah

Python dikembangkan oleh Giudo van Rossum, yaitu seorang programmer yang berasal dari Amsterdam, Belanda. Dipublikasikan pertama kali pada tahun 1990 dengan semboyan "Python is a programming language that lets you work quickly and integrate systems more effectively." Selanjutnya pada tahun 2000 terdapat pembaharuan versi Python ke versi 3.

Hingga saat ini Python masih dikembangkan oleh Python Software Foundation. Python versi 3 adalah versi yang paling banyak digunakan saat ini. Pemilihan nama Python sendiri diambil dari sebuah acara televisi yang lumayan terkenal yang bernama Mothy Python Flying Circus yang merupakan acara sirkus favorit dari Guido van Rossum.

Java

Java adalah bahasa pemrograman yang dapat dijalankan di berbagai perangkat komputer termasuk telepon genggam. Java banyak mengadopsi syntax yang terdapat pada C dan C++ namun dengan model object yang lebih sederhana. Java merupakan salah satu bahasa pemrograman yang bersifat umum (general purpose) yang mampu berjalan di berbagai platform melalui Java Virtual Machine (JVM). Solgan yang polpuler dari bahasa pemrograman Java ini yaitu "Write once, Run anywhere" yang artinya "Tulis sekali, Jalankan dimanapun". Saat ini java merupakan bahasa yang paling populer digunakan dalam pengembangan berbagai jenis perangkat lunak aplikasi seperti aplikasi native untuk smartphone dengan sistem operasi Android.

Java pun akrab dengan dunia saintifik dan akademik. Cukup banyak akademisi di Indonesia yang menggunakan Java sebagai alat bantu untuk menyelesaikan skripsi atau tugas akhir dengan berbagai topik yang didominasi kecerdasan buatan, data mining, enterprise architecture, aplikasi mobile, dan lainnya. Di dunia web development sendiri, Java memiliki berbagai web framework unggulan seperti Spring, Play Framework, Spark, Jakarta Struts, dan Java Server Pages.


Sejarah

Bahasa Java merupakan sebuah bahasa pemrograman yang legendaris. Hampir semua mahasiswa Informatika pasti pernah belajar dan membuat program sederhana menggunakan Java. Pada awalnya dibuat oleh James Gosling saat masih bergabung di Sun Microsystems yang saat ini merupakan bagian dari Oracle dan java ini dirilis pada tahun 1995.

Nama Java ini diambil dari kopi murni yang digiling langsung dari biji (kopi tubruk) kesukaan James Gosling. Konon kopi ini berasal dari Pulau Jawa. Jadi nama bahasa pemrograman Java tidak lain berasal dari kata Jawa (bahasa Inggris untuk Jawa adalah Java). Sejak saat itu logo atau icon dari bahasa pemrogaman java ditetapkan sebagai secangkir kopi yang terinspirasi dari "Java Coffee".

Sejak tahun 2010, Java diakuisisi dan diambil alih kepemilikannya oleh Oracle. Saat ini Java terus dikembangkan dan menjadi podasi bagi berbagai bahasa pemrograman lain seperti Kotlin, Scala, Groovy, JRuby, Jython, dan yang lainnya yang memanfaatkan Java Virtual Machine sebagai rumahnya. Menurut berbagai survey, Java selalu menempati posisi 5 besar dari bahasa pemrograman yang paling banyak digunakan di dunia.

PHP

PHP: Hypertext Preprocessor merupakan bahasa pemrograman server-side. PHP disebut bahasa pemrograman server side karena PHP diproses pada komputer server. Hal ini berbeda dibandingkan dengan bahasa pemrograman client-side seperti JavaScript yang diproses pada web browser (client). Biasanya digunakan dalam pengembangan sistem manajemen konten (Content Manajemen System) seperti Wordpress, Joomla, dan Drupal.

Selain itu juga sering digunakan untuk pengembangan back-end karena menyediakan fitur-fitur yang dapat mengakomodasi proses transfer informasi dari dan ke dalam database. Umumnya ditanamkan atau disisipkan ke dalam HTML untuk memprogram situs web dinamis. Selain itu, terdapat juga beberapa framework PHP yang terkenal seperti Laravel, Codigniter, Phalcon, Symfoni yang saat ini sedang populer dan banyak tersedia di internet untuk memudahkan proses pengembangan website menggunakan bahasa pemrograman PHP. Mayoritas website modern yang ada saat ini menggunakan framework PHP.

Sebelum ada PHP, pengembang kebanyakan menggunakan bahasa pemrograman HTML. Bahasa pemrograman ini sudah dapat membuat tampilan website, akan tetapi tampilan yang dihasilkan masih biasa saja. Biasa saja karena website masih bersifat statis atau tetap, tidak bisa menyesuaikan tampilan dengan kondisi pengguna. Semenjak adanya PHP maka pembuatan website dapat dikembangkan dan diatur agar menjadi website yang dinamis. Dimana website dapat menyesuaikan tampilan sesuai dengan perangkat yang digunakan oleh pengguna. Tidak hanya itu, website juga dapat melakukan input output dengan database.


Sejarah

Pada awalnya PHP merupakan singkatan dari Personal Home Page. Dibuat pertama kali oleh Rasmus Lerdorf pada tahun 1995. Pada waktu itu, PHP masih bernama Form Interpreted (FI) yang wujudnya berupa sekumpulan skrip yang digunakan untuk mengolah data formulir dari web. Selanjutnya Rasmus merilis kode sumber tersebut untuk umum dan menamakannya PHP/FI. Dengan perilisan kode sumber ini menjadi sumber terbuka, maka banyak pemrogram yang tertarik untuk ikut mengembangkan PHP.

Pada November 1997, dirilis PHP/FI 2.0. Pada rilis ini, interpreter PHP sudah diimplementasikan dalam program C. Dalam rilis ini disertakan juga modul-modul ekstensi yang meningkatkan kemampuan PHP/FI secara signifikan. Pada akhir tahun 1997, sebuah perusahaan bernama Zend menulis ulang interpreter PHP menjadi lebih bersih, lebih baik, dan lebih cepat. Kemudian pada Juni 1998, perusahaan tersebut merilis interpreter baru untuk PHP dan meresmikan rilis tersebut sebagai PHP 3.0 dan singkatan PHP diubah menjadi akronim berulang PHP: Hypertext Preprocessing.

Pada pertengahan tahun 1999, Zend merilis interpreter PHP baru dan rilis tersebut dikenal dengan PHP 4.0. PHP 4.0 adalah versi PHP yang paling banyak dipakai pada awal abad ke-21. Versi ini banyak dipakai disebabkan kemampuannya untuk membangun aplikasi web kompleks tetapi tetap memiliki kecepatan dan stabilitas yang tinggi.

Pada Juni 2004, Zend merilis PHP 5.0. Dalam versi ini, inti dari interpreter PHP mengalami perubahan besar. Versi ini juga memasukkan model pemrograman berorientasi objek ke dalam PHP untuk menjawab perkembangan bahasa pemrograman ke arah paradigma berorientasi objek. Peladen web bawaan ditambahkan pada versi 5.4 untuk mempermudah pengembang menjalankan kode PHP tanpa menginstal peladen perangkat lunak.

Versi terbaru dan stabil dari bahasa pemograman PHP saat ini adalah versi 7.0.16 dan 7.1.2 yang resmi dirilis pada tanggal 17 Februari 2017
